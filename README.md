# brief

## NPM
Frontend
```
npm i
```
Backend (critical for production)
```
cd server
npm i
```

### Run
Frontend
```
npm run serve
```
Backend
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
